Game Settings
If you double-click on the game settings at the top of the Project screen, you get this pretty pic.
Game Title: Self explanitory, right?

Main Script: Here you set the script that you want to run on startup.

Screen Width: The width of the screen you want your game to run in.

Screen Height: Take a guess. ^^

Click 'OK' when done.
When it asks if you want to use an empty tileset, you say yes.
